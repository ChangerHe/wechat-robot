# wechat-robot

一个基于wechaty的微信个人聊天机器人研究project

## 安装

```
// 复制项目到本地
$ git clone git@github.com:ChangerHe/wechat-robot.git
// 进入项目
$ cd wechat-robot
// 安装依赖
$ npm install
```

以上步骤可以将我的项目完整克隆到你的本地

## 启动测试

```
npm run demo
```

以上命令可以运行我的一个小demo, 用于自己测试

## 使用方法

微信扫描控制台打印的二维码, 点击登陆, 即可看到控制台打印的自己的微信的实时消息
